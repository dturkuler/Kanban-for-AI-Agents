# Contributing to Kanban for AI Agents

Thank you for your interest in contributing! We welcome all contributions to help make AI agent project management better.

## How to Contribute

1.  **Fork the Repository**: Create your own copy of the code.
2.  **Create a Branch**: Make a feature branch for your changes.
3.  **Commit Changes**: Use descriptive commit messages.
4.  **Push and PR**: Push your changes and submit a Pull Request.

## Development Setup

1.  Clone the repository.
2.  Navigate to `kanban-for-ai`.
3.  Run `npm install` to install dependencies.
4.  Run `npm run compile` to build the extension.
5.  Press `F5` in VS Code to launch the Extension Development Host.

## Code Style

-   Use TypeScript for all extension code.
-   Follow the existing formatting (Prettier/ESLint).

## Reporting Issues

Please use the GitHub Issues tab to report bugs or suggest features.
